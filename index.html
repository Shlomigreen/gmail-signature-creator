How to Add to Gmail:

  1. Copy your signature above using the "Copy Signature" button
  2. Go to Gmail Settings (gear icon → "See all settings")
  3. Scroll to "Signature" section
  4. Click "Create new" and give it a name
  5. In the signature editor, paste using Ctrl+V (Windows) or Cmd+V (Mac)
  6. If formatting is lost, try pasting with Ctrl+Shift+V then reformat manually
  7. For images: You may need to re-upload images if they don't appear
  8. Save changes and select as default signature

Alternative methods:

  • Manual Copy: Select the signature above and copy it with Ctrl+C/Cmd+C
  • HTML Code: Use "Copy HTML Code" button and paste into Gmail's HTML editor (Advanced)
